Introduction

This section covers the main integration techniques useful for applied mathematics in chemistry: substitution, integration by parts, and partial fraction decomposition.

1. Substitution (Change of variable)

Substitution simplifies an integral by letting \( u = g(x) \) with \( du = g'(x) dx \). Then the integral becomes: \[ \int f(x) dx = \int f(g^{-1}(u)) du \]

Example: Compute \(\int 2x \cos(x^2) dx\)

Let \( u = x^2 \) → \( du = 2x dx \). Then: \[ \int 2x \cos(x^2) dx = \int \cos(u) du = \sin(u) + C = \sin(x^2) + C \]

2. Integration by parts

Formula: \[ \int u \, dv = uv - \int v \, du \] where \( u = u(x) \) and \( dv = v'(x) dx \)

Example: Compute \(\int x e^x dx\)

Let \( u = x \Rightarrow du = dx\), and \( dv = e^x dx \Rightarrow v = e^x \) Then: \[ \int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C = e^x(x-1) + C \]

3. Partial fraction decomposition

This method simplifies integrals of rational functions \( \frac{P(x)}{Q(x)} \) where \( \deg P < \deg Q \). We write: \[ \frac{P(x)}{Q(x)} = \frac{A}{x-a} + \frac{B}{x-b} + \dots \]

Example: Compute \(\int \frac{1}{x^2 -1} dx\)

Factor: \(x^2 -1 = (x-1)(x+1)\) Decomposition: \(\frac{1}{x^2 -1} = \frac{A}{x-1} + \frac{B}{x+1}\) Solve: \(A = \frac{1}{2}, B = -\frac{1}{2}\) Integral: \[ \int \frac{1}{x^2 -1} dx = \frac{1}{2} \ln|x-1| - \frac{1}{2} \ln|x+1| + C \]

Exercises

  • Compute \(\int x^3 \sqrt{x^4 +1} dx\) (substitution)
  • Compute \(\int \ln(x) dx\) (integration by parts)
  • Compute \(\int \frac{2x+3}{x^2 +3x +2} dx\) (partial fractions)

Solutions

  • Substitution: \( u = x^4+1 \Rightarrow du = 4x^3 dx \Rightarrow \frac{1}{4} \int \sqrt{u} du = \frac{(x^4+1)^{3/2}}{6} + C \)
  • Integration by parts: \( \int \ln(x) dx = x \ln(x) - x + C \)
  • Partial fractions: \( \frac{2x+3}{x^2+3x+2} = \frac{1}{x+1} + \frac{1}{x+2} \Rightarrow \int \frac{1}{x+1} dx + \int \frac{1}{x+2} dx = \ln|x+1| + \ln|x+2| + C \)
