TD — Exercice 1 (suite)

Énoncé : Calculer les intégrales indéfinies suivantes :

  1. $I_1 = \displaystyle\int \frac{dx}{x(1+\ln^2 x)}$
  2. $I_2 = \displaystyle\int \arctan x\,dx$
  3. $I_3 = \displaystyle\int \ln(1+x^2)\,dx$
  4. $I_4 = \displaystyle\int \frac{dx}{2+e^x}$
  5. $I_5 = \displaystyle\int \frac{x\,dx}{(x^2+1)^{1/3}}$
  6. $I_6 = \displaystyle\int \frac{x+1}{(x+2)(x+3)}\,dx$
Solution complète

$I_1 = \displaystyle\int \frac{dx}{x(1+\ln^2 x)}$

Substitution $t = \ln x$ donc $dt = \frac{dx}{x}$.

\\[ I_1 = \int \frac{dt}{1+t^2} = \arctan t + C = \arctan(\ln x) + C. \\]

$I_2 = \displaystyle\int \arctan x\,dx$

Intégration par parties : $u = \arctan x$, $dv = dx$. Alors $du = \frac{dx}{1+x^2}$ et $v = x$.

\\[ I_2 = x\arctan x - \int \frac{x}{1+x^2}\,dx. \\]

Substitution $w = 1+x^2$, $dw = 2x\,dx$ :

\\[ \int \frac{x}{1+x^2}\,dx = \frac{1}{2}\ln(1+x^2). \\]

Donc $I_2 = x\arctan x - \tfrac{1}{2}\ln(1+x^2) + C.$

$I_3 = \displaystyle\int \ln(1+x^2)\,dx$

Intégration par parties : $u = \ln(1+x^2)$, $dv = dx$ donc $du = \frac{2x}{1+x^2}\,dx$, $v = x$.

\\[ I_3 = x\ln(1+x^2) - 2\int \frac{x^2}{1+x^2}\,dx. \\]

Or $\frac{x^2}{1+x^2} = 1 - \frac{1}{1+x^2}$. Donc

\\[ I_3 = x\ln(1+x^2) - 2x + 2\arctan x + C. \\]

$I_4 = \displaystyle\int \frac{dx}{2+e^x}$

Posons $t = e^x$, donc $dt = e^x dx = t\,dx$, ainsi $dx = \frac{dt}{t}$ :

\\[ I_4 = \int \frac{1}{t(2+t)}\,dt. \\]

Décomposition en éléments simples :

\\[ \frac{1}{t(2+t)} = \frac{1}{2}\left(\frac{1}{t} - \frac{1}{t+2}\right). \\]

Donc

\\[ I_4 = \tfrac{1}{2}\ln\left|\frac{t}{t+2}\right| + C = \tfrac{1}{2}\ln\left(\frac{e^x}{e^x+2}\right) + C. \\]

$I_5 = \displaystyle\int \frac{x\,dx}{(x^2+1)^{1/3}}$

Posons $u = x^2 + 1$, $du = 2x\,dx$, donc $x\,dx = \frac{1}{2}du$.

\\[ I_5 = \frac{1}{2}\int u^{-1/3}\,du = \frac{1}{2}\cdot\frac{u^{2/3}}{2/3} + C = \frac{3}{4}(x^2 + 1)^{2/3} + C. \\]

$I_6 = \displaystyle\int \frac{x+1}{(x+2)(x+3)}\,dx$

On cherche $A,B$ tels que

\\[ \frac{x+1}{(x+2)(x+3)} = \frac{A}{x+2} + \frac{B}{x+3}. \\]

En développant : $x+1 = A(x+3) + B(x+2)$, donc

$A + B = 1$ et $3A + 2B = 1$. Résolution : $A = -1$, $B = 2$.

\\[ I_6 = \int\left(-\frac{1}{x+2} + \frac{2}{x+3}\right)dx = -\ln|x+2| + 2\ln|x+3| + C. \\]

Remarque : Dans toutes les expressions, $C$ est une constante d’intégration.
