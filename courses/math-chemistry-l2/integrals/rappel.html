Rappel de cours : Techniques d’intégration

1. Intégration par parties

Formule principale :

$$\int u\,dv = uv - \int v\,du$$

Cette formule est très utile lorsqu’une fonction est un produit de deux autres fonctions. Elle provient directement de la formule de dérivation du produit :

$$(uv)' = u'v + uv'$$

En intégrant les deux membres :

$$\int (uv)'dx = \int u'v\,dx + \int uv'\,dx$$

Or, \(\int (uv)'dx = uv\), d’où la relation :

$$\int u\,v'\,dx = uv - \int v\,u'\,dx$$

En posant \(dv = v'\,dx\) et \(du = u'\,dx\), on obtient la formule classique : $$\int u\,dv = uv - \int v\,du$$

Exemple :

Calculons \( I = \int \arctan(x)\,dx \)

On pose :

  • \(u = \arctan(x)\) ⇒ \(du = \dfrac{1}{1+x^2}dx\)
  • \(dv = dx\) ⇒ \(v = x\)
$$I = x\arctan(x) - \int \frac{x}{1+x^2}dx$$

Or \(\int \frac{x}{1+x^2}dx = \frac{1}{2}\ln(1+x^2)\), donc :

$$\boxed{I = x\arctan(x) - \frac{1}{2}\ln(1+x^2) + C}$$

2. Changement de variable

Le changement de variable permet de simplifier une intégrale en remplaçant \(x\) par une nouvelle variable \(u\).

$$\int f(g(x))g'(x)\,dx = \int f(u)\,du \quad \text{où } u = g(x)$$

L’idée est simple : on cherche à reconnaître une dérivée g'(x) à côté d’une fonction composée f(g(x)).

Exemple :

Calculons \( I = \int \frac{dx}{x(1+\ln^2 x)} \)

On pose \( u = \ln x \) ⇒ \( du = \frac{dx}{x} \).

L’intégrale devient :

$$I = \int \frac{du}{1+u^2} = \arctan(u) + C = \boxed{\arctan(\ln x) + C}$$

3. Intégrales de fonctions rationnelles

Une fonction rationnelle est un quotient de deux polynômes : $$f(x) = \frac{P(x)}{Q(x)}.$$

Pour intégrer, on cherche à la décomposer en éléments simples :

$$ \frac{P(x)}{Q(x)} = \sum \frac{A_i}{(x - a_i)^{n_i}} + \sum \frac{B_jx + C_j}{(x^2 + b_jx + c_j)^{m_j}} $$

Chaque facteur irréductible du dénominateur donne un terme :

  • \((x-a)\) : facteur linéaire (racine réelle)
  • \((x^2+bx+c)\) : facteur quadratique irréductible sur ℝ

Exemple général :

$$\frac{x^8 + 2x^2 + 3x + 4}{(x^2 + 1)^2(x + 1)^2(x - 2)}$$

Décomposition générale :

$$ \frac{Ax + B}{x^2 + 1} + \frac{Cx + D}{(x^2 + 1)^2} + \frac{E}{x + 1} + \frac{F}{(x + 1)^2} + \frac{G}{x - 2} $$

Exemple avec division euclidienne :

Si le degré du numérateur ≥ degré du dénominateur, on commence par la division : 

Dividende :  A(x) = x³ + 2x² + 3x + 4
Diviseur :   B(x) = x² + 1

Étape 1 :  quotient partiel = x      car x³ ÷ x² = x
Soustraction : (x³ + 2x² + 3x + 4) - x(x² + 1) = 2x² + 2x + 4

Étape 2 :  quotient partiel = 2      car 2x² ÷ x² = 2
Soustraction : (2x² + 2x + 4) - 2(x² + 1) = 3x + 2

Reste : 3x + 2

Résultat :

$$\frac{x³ + 2x² + 3x + 4}{x² + 1} = (x + 2) + \frac{3x + 2}{x² + 1}$$

4. Intégrales trigonométriques

Pour simplifier les puissances de sinus et cosinus, on utilise les identités :

$$\sin^2 x = \frac{1 - \cos(2x)}{2}, \quad \cos^2 x = \frac{1 + \cos(2x)}{2}$$

Exemples :

Exemple 1 :

$$\int \sin^2 x\,dx = \frac{1}{2}\int (1 - \cos(2x))\,dx = \frac{x}{2} - \frac{\sin(2x)}{4} + C$$

Exemple 2 :

$$\int \cos^2 x\,dx = \frac{1}{2}\int (1 + \cos(2x))\,dx = \frac{x}{2} + \frac{\sin(2x)}{4} + C$$
