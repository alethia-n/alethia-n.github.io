Rappels – Intégrales

1️⃣ Définition

L’intégrale de \(f(x)\) sur [a,b] représente l’aire sous la courbe :

\\(\int_a^b f(x) dx\\)

2️⃣ Primitives usuelles

  • \\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\\)
  • \\(\int e^{kx} dx = \frac{1}{k} e^{kx} + C\\)
  • \\(\int \sin(x) dx = -\cos(x) + C\\)
  • \\(\int \frac{dx}{\sqrt{1-x^2}} = \arcsin(x) + C\\)

3️⃣ Composition de fonctions

Exemples :

  • \\(h(x) = \sin(x^2) = f(g(x))\\)
  • \\(h(x) = e^{3x+1} = f(g(x))\\)
  • \\(h(x) = \ln(2x^2+5) = f(g(x))\\)

4️⃣ Décomposition en éléments simples

\\(\frac{2x+3}{x^2+3x+2} = \frac{1}{x+1} + \frac{1}{x+2}\\)

\\(\int \frac{2x+3}{x^2+3x+2} dx = \ln|x+1| + \ln|x+2| + C\\)

5️⃣ Division euclidienne

Dividende : x^2 + 3x + 2
Diviseur  : x + 1
1. x^2 ÷ x = x
2. x*(x+1) = x^2 + x
3. Soustraction : (x^2+3x+2)-(x^2+x)=2x+2
4. 2x ÷ x = 2
5. 2*(x+1)=2x+2
6. Soustraction = 0
Quotient : x+2, Reste : 0
