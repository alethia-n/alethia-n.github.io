Integrals – Notes

1️⃣ Definition

The integral of a function \(f(x)\) over an interval \([a,b]\) represents the area under the curve:

\[ \int_a^b f(x)\,dx \]

2️⃣ Standard Primitives

  • \(\int x^n dx = \frac{x^{n+1}}{n+1} + C\), \(n\neq -1\)
  • \(\int e^{kx} dx = \frac{1}{k} e^{kx} + C\)
  • \(\int \sin(x) dx = -\cos(x) + C\)
  • \(\int \cos(x) dx = \sin(x) + C\)
  • \(\int \frac{dx}{1+x^2} = \arctan(x) + C\)
  • \(\int \frac{dx}{\sqrt{1-x^2}} = \arcsin(x) + C\)

3️⃣ Composition of Functions

For a composite function \(f(g(x))\), the integral can use substitution:

\[ \int f(g(x)) g'(x)\,dx = \int f(u)\,du, \quad u=g(x) \]

Example:

\[ \int 2x \cos(x^2) dx = \int \cos(u) du = \sin(u) + C = \sin(x^2) + C \]

4️⃣ Integration Techniques

4.1 Substitution Method

Use when the integrand contains a composition \(f(g(x)) g'(x)\). Example:

\[ \int x e^{x^2} dx = \frac{1}{2} e^{x^2} + C \]

4.2 Integration by Parts

Formula:

\[ \int u\, dv = u v - \int v\, du \]
\[ \int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C \]

4.3 Partial Fraction Decomposition

Use for rational functions \(\frac{P(x)}{Q(x)}\), deg(P) < deg(Q). If deg(P) ≥ deg(Q), first do Euclidean division.

\[ \frac{2x+3}{x^2+3x+2} = \frac{1}{x+1} + \frac{1}{x+2} \] \[ \int \frac{2x+3}{x^2+3x+2} dx = \ln|x+1| + \ln|x+2| + C \]

4.4 Euclidean Division Example

Divide \(x^3 + 2x^2 + 3x + 4\) by \(x^2 + x + 1\):

Step 1: x^3 ÷ x^2 = x
Step 2: Multiply: x*(x^2 + x +1) = x^3 + x^2 + x
Step 3: Subtract: (x^3 +2x^2 +3x +4) - (x^3 + x^2 + x) = x^2 + 2x + 4
Step 4: Divide: x^2 ÷ x^2 = 1
Step 5: Multiply: 1*(x^2 + x +1) = x^2 + x +1
Step 6: Subtract: (x^2 + 2x +4) - (x^2 + x +1) = x +3
Quotient: x +1, Remainder: x +3
