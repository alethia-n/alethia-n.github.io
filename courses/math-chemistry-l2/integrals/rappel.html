1. Intégration par parties

Cette méthode découle directement de la formule de dérivation du produit : \((uv)' = u'v + uv'\).

$$\int u\,dv = uv - \int v\,du$$

En posant \( du = u'(x)\,dx \) et \( dv = v'(x)\,dx \), la formule s’écrit :

$$\int u(x)v'(x)\,dx = u(x)v(x) - \int u'(x)v(x)\,dx$$

Remarque : On choisit u pour qu’il devienne plus simple lorsqu’on le dérive, et dv pour que son intégrale soit facile à calculer.

Exemple :

Calculons \( I = \int \arctan(x)\,dx \).

Posons : \(u = \arctan(x)\) ⇒ \(du = \frac{1}{1+x^2}\,dx\), \(dv = dx\) ⇒ \(v = x\)

$$I = x\arctan(x) - \int \frac{x}{1+x^2}\,dx = x\arctan(x) - \frac{1}{2}\ln(1+x^2) + C$$

2. Changement de variable

Cette méthode est utilisée lorsqu’une substitution simplifie l’expression sous le signe intégral.

$$\int f(g(x))\,g'(x)\,dx = \int f(u)\,du \quad \text{avec } u=g(x)$$

Exemple :

Calculons \( I = \int \frac{dx}{x(1+\ln^2x)} \).

Posons \( u = \ln(x) \) ⇒ \( du = \frac{dx}{x} \).

$$I = \int \frac{du}{1+u^2} = \arctan(u) + C = \arctan(\ln(x)) + C$$

3. Intégrales de fonctions rationnelles

Une fonction rationnelle est un quotient de deux polynômes : \( R(x) = \frac{P(x)}{Q(x)} \).

Composantes irréductibles :

  • Une composante irréductible de degré 1 : facteur linéaire du type \((x - a)\).
  • Une composante irréductible de degré 2 : polynôme quadratique sans racine réelle, du type \((x^2 + px + q)\) avec \(\Delta < 0\).

Toute fraction rationnelle peut être décomposée en somme d’éléments simples selon la nature de ces facteurs.

Cas où le degré du numérateur ≥ degré du dénominateur :

On effectue d’abord une division euclidienne :

$$P(x) = Q(x)\,D(x) + R(x), \quad \deg(R) < \deg(Q)$$

Exemple :

Soit \[ \frac{x^3 + 2x^2 + 3x + 4}{x^2 + 1}. \] On effectue la division euclidienne : 

Diviseur :   x² + 1
Dividende :  x³ + 2x² + 3x + 4

1re étape : (x³ + 2x² + 3x + 4) ÷ (x² + 1)
→ Quotient partiel : x → on soustrait (x³ + x)

Reste 1 : (2x² + 2x + 4)
→ Quotient partiel suivant : +2 → on soustrait (2x² + 2)

Reste final : 3x + 2

Donc : \[ \frac{x^3 + 2x^2 + 3x + 4}{x^2 + 1} = (x + 2) + \frac{3x + 2}{x^2 + 1} \]
