Rappels – Techniques d’intégration (L2)

1. Intégration par parties

Formule 1 (écriture en fonction de x) :

$$\displaystyle \int u(x)\,v'(x)\,dx = u(x)\,v(x) - \int u'(x)\,v(x)\,dx.$$

Formule 2 (écriture abrégée) :

$$\displaystyle \int u\,dv = uv - \int v\,du.$$

Comment relier les deux écritures :

Dans la première formule, tout est écrit en fonction de la variable \(x\). On note que \(u'(x)\) est la dérivée de \(u\), et \(v'(x)\) celle de \(v\).

Pour simplifier, on remplace :

  • \(du = u'(x)\,dx\),
  • \(dv = v'(x)\,dx\).

Ce ne sont pas de nouvelles notions : c’est simplement une manière plus courte d’écrire les mêmes choses. En remplaçant ces deux expressions dans la première formule, on obtient directement la deuxième :

$$\int u\,dv = uv - \int v\,du.$$

Cette formule vient de la règle de dérivation du produit :

$$(u v)' = u'v + uv'.$$

En intégrant les deux côtés, on obtient :

$$uv = \int u'v\,dx + \int uv'\,dx,$$

d’où :

$$\displaystyle \int uv'\,dx = uv - \int u'v\,dx.$$

Exemple :

$$\displaystyle I=\int \arctan x\,dx.$$

Choix pour l'intégration par parties :

On choisit habituellement \(u\) comme la partie « compliquée » qui simplifie lorsqu'on la dérive. Ici on prend :

  • \(u=\arctan x\) ⟶ \(du=\dfrac{1}{1+x^2}\,dx\)
  • \(dv=dx\) ⟶ \(v=x\)

Appliquer la formule d'intégration par parties :

$$\int u\,dv = uv - \int v\,du.$$

Donc

$$\begin{aligned} I &= \int \arctan x \, dx = x\arctan x - \int x\cdot\frac{1}{1+x^2}\,dx \\ &= x\arctan x - \int \frac{x}{1+x^2}\,dx. \end{aligned}$$

Calcul de la seconde intégrale :

Posons \(t = 1+x^2\) (ou remarquez que la dérivée de \(1+x^2\) est \(2x\)), alors

$$\int \frac{x}{1+x^2}\,dx = \frac{1}{2}\int \frac{2x}{1+x^2}\,dx = \frac{1}{2}\ln(1+x^2) + C.$$

Conclusion :

$$\boxed{\,\displaystyle \int \arctan x\,dx = x\arctan x - \frac{1}{2}\ln(1+x^2) + C \,}$$

Vérification (dérivation) :

$$\frac{d}{dx}\Big(x\arctan x - \tfrac{1}{2}\ln(1+x^2)\Big) = \arctan x + x\cdot\frac{1}{1+x^2} - \frac{1}{2}\cdot\frac{2x}{1+x^2} = \arctan x.$$

Le résultat est donc correct. Remarque : le choix \(u=\arctan x\) est naturel car sa dérivée est simple (\(1/(1+x^2)\)) et \(dv=dx\) a pour primitive \(v=x\), ce qui rend le calcul direct.

2. Changement de variable

Formule du changement de variable :

$$\displaystyle \int f(g(x))\,g'(x)\,dx = \int f(u)\,du$$

Idée générale :

Le changement de variable sert à simplifier une intégrale lorsqu’une partie de l’expression est la dérivée d’une autre. On remplace cette expression par une nouvelle variable \(u\) afin d’obtenir une intégrale plus simple.

Étapes de la méthode :

  1. Choisir une substitution \(u = g(x)\), souvent une fonction « intérieure » comme \(\ln x\), \(x^2\), \(\sin x\)…
  2. Calculer la dérivée : \(du = g'(x)\,dx.\)
  3. Remplacer dans l’intégrale : \(g(x)\) devient \(u\), et \(g'(x)\,dx\) devient \(du.\)
  4. Résoudre l’intégrale en \(u\).
  5. Revenir à la variable initiale \(x\) à la fin.

Exemple d’application

Calculer l’intégrale suivante :

$$\displaystyle I = \int \frac{dx}{x(1+\ln^2 x)}.$$

Étape 1 : on reconnaît la présence de \(\ln x\) et de sa dérivée \(\dfrac{1}{x}\). On choisit donc la substitution :

$$u = \ln x.$$

Étape 2 : on dérive :

$$du = \frac{1}{x}\,dx.$$

Étape 3 : on remplace dans l’intégrale :

$$I = \int \frac{dx}{x(1+\ln^2 x)} = \int \frac{du}{1+u^2}.$$

Étape 4 : on reconnaît une formule connue :

$$\int \frac{du}{1+u^2} = \arctan(u) + C.$$

Étape 5 : on revient à la variable \(x\) :

$$\boxed{I = \arctan(\ln x) + C.}$$

Vérification :

$$\frac{d}{dx}\big[\arctan(\ln x)\big] = \frac{1}{1+(\ln x)^2}\cdot\frac{1}{x} = \frac{1}{x(1+\ln^2 x)}.$$

Le résultat est donc correct. Ce calcul illustre parfaitement l’intérêt du changement de variable : il transforme une intégrale compliquée en une intégrale simple et connue.

Remarques :

  • Le changement de variable ne modifie pas la valeur de l’intégrale, il sert uniquement à la simplifier.
  • La relation \(du = g'(x)\,dx\) provient directement de la dérivation de \(u = g(x)\).
  • Pour une intégrale définie, il faut aussi changer les bornes :
    $$\int_a^b f(g(x))g'(x)\,dx = \int_{g(a)}^{g(b)} f(u)\,du.$$

3. Intégrales de fonctions rationnelles

Une fonction rationnelle est le quotient de deux polynômes :

$$R(x) = \frac{P(x)}{Q(x)}, \quad \text{où } P(x) \text{ et } Q(x) \text{ sont des polynômes, et } Q(x) \neq 0.$$

Pour intégrer ce type de fonction, on cherche souvent à la simplifier en la décomposant en éléments simples. C’est ce qu’on appelle la décomposition en éléments simples (ou « D.E.S. »).

1. Notion de fraction rationnelle et d’élément irréductible

Une fraction rationnelle est simplement une fraction dont le numérateur et le dénominateur sont des polynômes. Exemple :

$$\frac{x^2 + 3x + 2}{x^3 - x}.$$

Un élément irréductible (sur ℝ) est un polynôme qui ne peut pas être factorisé avec des coefficients réels. Par exemple :

  • \(x - a\) est un facteur linéaire (donc irréductible).
  • \(x^2 + bx + c\) est irréductible si son discriminant est négatif : \(b^2 - 4c < 0\).

Ainsi, tout dénominateur se factorise en produit de termes linéaires ou quadratiques irréductibles.

2. Comparaison des degrés

  • Cas 1 : si le degré du numérateur est inférieur à celui du dénominateur, la fraction est dite propre. On peut directement la décomposer.
  • Cas 2 : si le degré du numérateur est supérieur ou égal à celui du dénominateur, on doit d’abord effectuer une division euclidienne des polynômes.

3. Division euclidienne de deux polynômes

Soient \(P(x)\) et \(Q(x)\) deux polynômes. On peut écrire :

$$P(x) = Q(x)\,A(x) + R(x),$$

où :

  • \(A(x)\) est le quotient,
  • \(R(x)\) est le reste (degré strictement inférieur à celui de \(Q(x)\)).

Ainsi :

$$\frac{P(x)}{Q(x)} = A(x) + \frac{R(x)}{Q(x)}.$$

On pourra ensuite intégrer \(A(x)\) et \(\dfrac{R(x)}{Q(x)}\) séparément.

Exemple :

Diviser \(P(x) = x^3\) par \(Q(x) = x^2 + 1\).

On cherche \(A(x)\) et \(R(x)\) tels que :

$$x^3 = (x^2 + 1)A(x) + R(x).$$

On commence par :

  • le terme dominant de \(P\) est \(x^3\), de \(Q\) c’est \(x^2\), donc le premier terme du quotient est \(x\).

On soustrait :

$$x^3 - x(x^2 + 1) = -x.$$

Donc :

$$A(x) = x,\quad R(x) = -x.$$

Et ainsi :

$$\frac{x^3}{x^2 + 1} = x - \frac{x}{x^2 + 1}.$$

On pourra donc intégrer :

$$\int \frac{x^3}{x^2 + 1}\,dx = \int x\,dx - \int \frac{x}{x^2 + 1}\,dx.$$

Intégration : - \(\displaystyle \int x\,dx = \frac{x^2}{2}\) - Pour la seconde, on pose \(u = x^2 + 1\), \(du = 2x\,dx\), donc :

$$\int \frac{x}{x^2 + 1}\,dx = \frac{1}{2}\ln(x^2 + 1) + C.$$

Résultat final :

$$\boxed{\,\displaystyle \int \frac{x^3}{x^2 + 1}\,dx = \frac{x^2}{2} - \frac{1}{2}\ln(1+x^2) + C\,}$$

4. Cas d’une fraction déjà propre

Considérons maintenant un exemple où le numérateur est de degré plus petit que le dénominateur :

$$I = \int \frac{x}{x^2 + 1}\,dx.$$

Ici, le degré du numérateur (1) est inférieur à celui du dénominateur (2). On reconnaît directement une forme de dérivée : celle de \(x^2 + 1\).

Posons \(u = x^2 + 1 \Rightarrow du = 2x\,dx\).

$$I = \frac{1}{2}\int \frac{du}{u} = \frac{1}{2}\ln|u| + C = \frac{1}{2}\ln(1 + x^2) + C.$$

C’est donc une intégrale de type simple, sans besoin de division.

5. Méthode générale

  1. Vérifier le degré du numérateur et du dénominateur.
  2. Si le numérateur a un degré ≥ au dénominateur, faire une division euclidienne.
  3. Factoriser le dénominateur en éléments irréductibles.
  4. Décomposer en éléments simples (formes du type \(\dfrac{A}{x-a}\), \(\dfrac{Bx+C}{x^2+px+q}\)…).
  5. Intégrer chaque terme à l’aide de formules connues :
    • \(\displaystyle \int \frac{dx}{x-a} = \ln|x-a| + C\)
    • \(\displaystyle \int \frac{dx}{x^2 + 1} = \arctan x + C\)
    • \(\displaystyle \int \frac{x\,dx}{x^2 + 1} = \frac{1}{2}\ln(1+x^2) + C\)

Cette méthode permet de traiter toutes les intégrales rationnelles, quelles que soient leurs formes.

4. Intégrales trigonométriques

Utiliser les identités trigonométriques :

$$\sin^2 x = \frac{1 - \cos(2x)}{2}, \quad \cos^2 x = \frac{1 + \cos(2x)}{2}$$

5. Intégrales impropres

$$\int_a^{+\infty} f(x)\,dx = \lim_{b\to+\infty} \int_a^b f(x)\,dx$$
