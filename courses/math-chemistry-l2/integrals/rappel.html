Integrals – Notes

1️⃣ Definition

The integral of a function \(f(x)\) over [a,b] represents the area under the curve:

\[ \int_a^b f(x)\,dx \]

2️⃣ Standard Primitives

  • \(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)
  • \(\int e^{kx} dx = \frac{1}{k} e^{kx} + C\)
  • \(\int \sin(x) dx = -\cos(x) + C\)
  • \(\int \frac{dx}{\sqrt{1-x^2}} = \arcsin(x) + C\)
  • \(\int \frac{dx}{1+x^2} = \arctan(x) + C\)

3️⃣ Partial Fraction Decomposition

\[ \frac{2x+3}{x^2+3x+2} = \frac{1}{x+1} + \frac{1}{x+2} \] \[ \int \frac{2x+3}{x^2+3x+2} dx = \ln|x+1| + \ln|x+2| + C \]

4️⃣ Euclidean Division of Polynomials

Divide x^2 + 3x + 2 by x + 1:

Step 1: x^2 ÷ x = x
Step 2: x*(x+1) = x^2 + x
Step 3: Subtract: (x^2+3x+2)-(x^2+x)=2x+2
Step 4: 2x ÷ x = 2
Step 5: 2*(x+1)=2x+2
Step 6: Subtract: 0

Quotient: x+2, Remainder: 0
