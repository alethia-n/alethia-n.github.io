Rappels : techniques d'intégration (niveau L2)

Cette page rassemble les méthodes usuelles, formules importantes et exemples-clefs pour les intégrales simples, impropres et multiples (double intégrale, changement de variables). Pour chaque technique : principe, quand l'utiliser, exemples.

1. Définitions & rappels essentiels

  • Primitive : F est une primitive de f sur I si F'(x)=f(x). Les primitives diffèrent par une constante : F+C.
  • Intégrale indéfinie : notation ∫ f(x) dx — ensemble des primitives.
  • Intégrale définie : ∫_a^b f(x) dx = F(b)-F(a) (Théorème fondamental de l'analyse).
  • Intégrale impropre : une borne infinie ou une singularité dans l'intervalle → définition par limite (convergence / divergence).

2. Tableau de primitives usuelles (rapide)

∫ x^n dx = x^(n+1)/(n+1)   (n ≠ -1)
∫ dx/x = ln|x| + C
∫ e^x dx = e^x + C
∫ a^x dx = a^x / ln a + C    (a>0, a≠1)
∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
∫ 1/(1+x^2) dx = arctan x + C
∫ 1/√(a^2 - x^2) dx = arcsin(x/a) + C

3. Méthode de la substitution (chang. de variable)

Principe : si u = g(x) et du = g'(x) dx, on transforme ∫ f(g(x)) g'(x) dx en ∫ f(u) du. Utilisée partout : compositions, racines, puissances, trig.

Exemple :

∫ 2x cos(x^2) dx. Poser u = x^2 ⇒ du = 2x dx ⇒ ∫ cos(u) du = sin u + C = sin(x^2)+C

4. Intégration par parties

Formule : ∫ u dv = u v − ∫ v du.

Conseil : choisir u = partie qui se simplifie quand on dérive (log, polynôme), dv = le reste.

Exemple :

∫ x e^x dx. Poser u=x (u'=1), dv=e^x dx (v=e^x)
⇒ ∫ x e^x dx = x e^x − ∫ e^x dx = x e^x − e^x + C = e^x(x−1)+C

Technique tabulaire : répéter parties pour polynôme × exponentielle/trigonométrique.

5. Intégrales trigonométriques

  • sin^m x cos^n x :
    • si un exposant est impair → extraire un facteur sin x ou cos x et faire u = cos x ou u = sin x.
    • si les deux sont pairs → utiliser formules d'angle double : sin^2 = (1−cos2x)/2, cos^2 = (1+cos2x)/2, et réduire.
  • tan^n x sec^m x :
    • si m pair → extraire sec^2 x et poser u = tan x.
    • si n impair → extraire tan x sec x et poser u = sec x.
  • Weierstrass (t = tan(x/2)) : utile pour rationaliser sin, cos, dx : 
    sin x = 2t/(1+t^2),  cos x = (1−t^2)/(1+t^2),  dx = 2/(1+t^2) dt

6. Rationales : intégration de fonctions rationnelles

Procédure :

  1. Si deg(num) ≥ deg(den) → division polynomiale.
  2. Factoriser dénominateur au degré 1 (linéaire) ou 2 irréductible (quadratique).
  3. Décomposer en éléments simples (fractions partielles).
  4. Intégrer chaque terme : ln pour pôles simples, arctan pour quadratiques irréductibles, etc.

Exemple :

∫ dx/(x^2 − 1) = 1/2 ∫ (1/(x−1) − 1/(x+1)) dx = 1/2 [ln|x−1| − ln|x+1|] + C

7. Substitutions trigonométriques (pour racines)

Cas typiques :

  • √(a^2 − x^2) → poser x = a sin θ
  • √(a^2 + x^2) → poser x = a tan θ
  • √(x^2 − a^2) → poser x = a sec θ

Exemple : 

∫ dx / √(a^2 − x^2). Poser x = a sin θ ⇒ dx = a cos θ dθ, √(a^2 − x^2) = a cos θ
⇒ ∫ (a cos θ)/(a cos θ) dθ = ∫ dθ = θ + C = arcsin(x/a) + C

8. Intégrales impropres — tests de convergence

Deux types : borne infinie (ex: ∫_1^∞) ou singularité interne (ex: ∫_0^1 1/√x dx). On définit par des limites.

Tests usuels :

  • Comparaison : si 0 ≤ f ≤ g et ∫ g converge ⇒ ∫ f converge.
  • Comparaison limite : si lim f/g = L ∈ (0, ∞) alors f et g ont même nature (conv/div).
  • p-intégrales : ∫_1^∞ 1/x^p converge ⇔ p>1 ; ∫_0^1 1/x^p converge ⇔ p<1.
  • Convergence absolue : si ∫ |f| converge alors ∫ f converge (mais pas réciproque).

9. Intégration numérique (approximations)

Quand la primitive n'est pas élémentaire :

  • Riemann : définition → somme de moyennes × largeur.
  • Trapèze : assez simple, erreur liée à f''(ξ).
  • Simpson : meilleure précision pour polynômes de degré ≤ 3, erreur ~ f^{(4)}.

Donner les formules si tu veux les implémentations en TD.

10. Intégrales multiples (double intégrale)

Fubini : si f est continue sur rectangle, ∫∫_R f(x,y) dA = ∫ (∫ f(x,y) dx) dy = ∫ (∫ f(x,y) dy) dx.

Coordonnées polaires : x = r cos θ, y = r sin θ, et dA = r dr dθ. Très utile pour régions circulaires.

Ex : aire d'un disque de rayon R = ∫_{θ=0}^{2π} ∫_{r=0}^R r dr dθ = π R^2

11. Changement de variable pour intégrales multiples (Jacobian)

Si (x,y) = Φ(u,v), alors dA = |det DΦ(u,v)| du dv. Pour le passage cartésien→polaire, det = r.

12. Fonctions spéciales (aperçu)

Gamma : Γ(s) = ∫_0^∞ t^{s-1} e^{-t} dt (prolonge factorielle).

Beta : B(p,q) = ∫_0^1 t^{p-1} (1−t)^{q-1} dt, et B(p,q)=Γ(p)Γ(q)/Γ(p+q).

Utiles en intégrales paramétriques et calcul d'intégrales définies.

13. Astuces pratiques & checklist

Checklist avant de commencer :
  1. Simplifier l'expression (factoriser, développer si nécessaire).
  2. Regarder si c'est une dérivée “facile” (substitution immédiate).
  3. Tester une substitution u=g(x).
  4. Tester parties (pour produit polynôme × exp/trig/log).
  5. Pour rationnelles, envisager décomposition en fractions partielles.
  6. Pour racines, envisager substitution trigonométrique.
  7. Pour intégrales sur R ou avec singularités → penser aux tests de convergence.

14. Exemples rapides résolus

Ex. 1 : ∫ x^2 ln x dx. Poser u = ln x → du = dx/x ; dv = x^2 dx → v = x^3/3 :

∫ x^2 ln x dx = (ln x)(x^3/3) − ∫ (x^3/3) * (1/x) dx
= x^3/3 ln x − ∫ x^2/3 dx = x^3/3 ln x − x^3/9 + C

Ex. 2 : ∫_1^∞ 1/(x (ln x)^2) dx. Poser t = ln x ⇒ dt = dx/x, limites : ln1=0 → ∞ ⇒ ∫_0^∞ 1/t^2 dt diverge en 0 ; mais attention à l'orientation : en fait ∫_2^∞ etc. (vérifier bornes avant de conclure).

15. Table de primitives / formules utiles (annexe rapide)

∫ cos^n x dx, ∫ sin^n x dx (règle selon n pair/impair)
Formules d'angle double/puissance :
  sin^2 x = (1−cos 2x)/2 ; cos^2 x = (1+cos 2x)/2
Intégrales usuelles pour fractions partielles : ∫ dx/(x−a) = ln|x−a|
                                            ∫ dx/(x^2 + a^2) = 1/a arctan(x/a)
