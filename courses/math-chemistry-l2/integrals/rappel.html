Rappels – Intégrales Niveau L2 Chimie

1️⃣ Définition de l’intégrale

L’intégrale d’une fonction \(f(x)\) sur un intervalle \([a,b]\) représente l’aire sous la courbe de \(f(x)\) entre \(x = a\) et \(x = b\).

Notation : \\(\int_a^b f(x) dx\\)

Intégrale indéfinie : \\(\int f(x) dx = F(x) + C\\), où \(F(x)\) est une primitive et \(C\) la constante d’intégration.

2️⃣ Primitives des fonctions usuelles

  • Polynomiales : \\(\int x^n dx = \frac{x^{n+1}}{n+1} + C, n \neq -1\\)
  • Exponentielles : \\(\int e^{kx} dx = \frac{1}{k} e^{kx} + C\\)
  • Trigonométriques : \\(\int \sin(x) dx = -\cos(x) + C, \int \cos(x) dx = \sin(x) + C\\)
  • Trigonométriques inverses : \\(\int \frac{dx}{\sqrt{1-x^2}} = \arcsin(x)+C, \int \frac{dx}{1+x^2} = \arctan(x)+C\\)

3️⃣ Notion de composition de fonctions

Une fonction \(h(x)\) peut être écrite comme composition de deux fonctions \(f\) et \(g\) :

\\(h(x) = f(g(x))\\)

Exemples :

  • \\(h(x) = \sin(x^2) \Rightarrow f(u)=\sin(u), g(x)=x^2\\)
  • \\(h(x) = e^{3x+1} \Rightarrow f(u)=e^u, g(x)=3x+1\\)
  • \\(h(x) = \ln(2x^2+5) \Rightarrow f(u)=\ln(u), g(x)=2x^2+5\\)

4️⃣ Décomposition en éléments simples

Pour un rationnel \\(\frac{P(x)}{Q(x)}\\) avec \\(\deg(P)<\deg(Q)\\), on l’écrit comme somme de fractions simples faciles à intégrer.

Exemple :

\\(\frac{2x+3}{x^2+3x+2} = \frac{2x+3}{(x+1)(x+2)} = \frac{A}{x+1} + \frac{B}{x+2}\\)

Multiplier par \\((x+1)(x+2)\\) : \\(2x+3 = A(x+2) + B(x+1)\\)

Identification : \\(A=1, B=1\\)

Intégrale : \\(\int \frac{2x+3}{x^2+3x+2} dx = \ln|x+1| + \ln|x+2| + C\\)

5️⃣ Division euclidienne de polynômes

Cas \\(\deg(P) \ge \deg(Q)\\) : on écrit

\\(P(x) = Q(x) \cdot S(x) + R(x), \deg(R)<\deg(Q)\\)

Alors :

\\(\frac{P(x)}{Q(x)} = S(x) + \frac{R(x)}{Q(x)}\\)

Intégration :

\\(\int \frac{P(x)}{Q(x)} dx = \int S(x) dx + \int \frac{R(x)}{Q(x)} dx\\)

Exemple détaillé :

\\(\int \frac{x^2+3x+2}{x+1} dx\\)

Étape 1 : Division

Dividende : x^2 + 3x + 2
Diviseur  : x + 1

1. Diviser le terme dominant : x^2 ÷ x = x → quotient partiel
2. Multiplier : x*(x+1) = x^2 + x
3. Soustraire : (x^2 + 3x + 2) - (x^2 + x) = 2x + 2
4. Diviser terme dominant : 2x ÷ x = 2 → ajouter au quotient
5. Multiplier : 2*(x+1) = 2x + 2
6. Soustraire : (2x + 2) - (2x + 2) = 0

Quotient : \\(S(x)=x+2\\), reste : \\(R(x)=0\\)

Donc : \\(\frac{x^2+3x+2}{x+1} = x+2\\)

Intégration : \\(\int \frac{x^2+3x+2}{x+1} dx = \int (x+2) dx = \frac{x^2}{2}+2x+C\\)

6️⃣ Exemples supplémentaires

  • Substitution trigonométrique : \\(\int \frac{dx}{\sqrt{a^2-x^2}} = \arcsin\left(\frac{x}{a}\right) + C\\)
  • Intégrale exponentielle : \\(\int e^{-kx} dx = -\frac{1}{k} e^{-kx} + C\\)
  • Intégrale rationnelle avec racine quadratique : \\(\int \frac{dx}{x^2+2x+5} = \frac{1}{2} \arctan\left(\frac{x+1}{2}\right) + C\\)
