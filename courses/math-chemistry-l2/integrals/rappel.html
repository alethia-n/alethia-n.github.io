Rappel de cours — Techniques d’intégration (complet)

1. Intégration par parties

Forme usuelle (en \(x\)) :

$$\displaystyle \int u(x)\,v'(x)\,dx = u(x)v(x) - \int u'(x)\,v(x)\,dx.$$

Forme abrégée (différentielle) :

$$\displaystyle \int u\,dv = uv - \int v\,du.$$

Pourquoi on écrit \(du = u'(x)\,dx\) ? (explication simple et rigoureuse)

Si \(u\) est dérivable en \(x\), sa dérivée est \(\dfrac{du}{dx}=u'(x)\). Par définition, pour un petit accroissement \(\Delta x\) on a :

$$u(x+\Delta x)-u(x) = u'(x)\,\Delta x + o(\Delta x).$$

En posant \(dx=\Delta x\) et \(du=u(x+\Delta x)-u(x)\), on obtient (au premier ordre)

$$du = u'(x)\,dx.$$

C'est donc une écriture compacte qui signifie « la variation linéaire de \(u\) associée à la variation \(dx\) de \(x\) ».

Origine de la formule

$$(u v)' = u'v + uv'$$

Intégrer des deux côtés et isoler la partie désirée donne la formule.

Exemple (niveau L2) : \(\displaystyle I=\int \arctan x\,dx\)

Choix : prendre \(u=\arctan x\) (devient plus simple en la dérivant), \(dv=dx\).

  • \(u=\arctan x \Rightarrow du=\dfrac{1}{1+x^2}\,dx\)
  • \(dv=dx \Rightarrow v=x\)
$$\begin{aligned} I &= x\arctan x - \int x\cdot\frac{1}{1+x^2}\,dx \\ &= x\arctan x - \frac{1}{2}\int \frac{2x}{1+x^2}\,dx = x\arctan x - \frac{1}{2}\ln(1+x^2) + C. \end{aligned}$$

2. Changement de variable

$$\displaystyle \int f(g(x))\,g'(x)\,dx = \int f(u)\,du \quad\text{où } u=g(x).$$

Étapes : choisir \(u=g(x)\), calculer \(du=g'(x)\,dx\), remplacer et intégrer en \(u\).

Exemple demandé (niveau L2) :

Calcul de :

$$\displaystyle I=\int \frac{dx}{x(1+\ln^2 x)}$$

Substitution : \(u=\ln x \Rightarrow du=\dfrac{dx}{x}\).

$$I=\int\frac{du}{1+u^2}=\arctan(u)+C=\boxed{\arctan(\ln x)+C}.$$

Vérification facile par dérivation : \(\dfrac{d}{dx}\arctan(\ln x)=\dfrac{1}{1+\ln^2 x}\cdot\dfrac{1}{x}\).

3. Intégrales de fonctions rationnelles — décomposition en éléments simples (complet)

Définition

Une fraction rationnelle est un quotient de deux polynômes :

$$R(x)=\dfrac{P(x)}{Q(x)},\qquad Q(x)\not\equiv 0.$$

Composantes irréductibles (sur \(\mathbb{R}\))

Le polynôme \(Q(x)\) se factorise (sur \(\mathbb{R}\)) en facteurs linéaires et quadratiques irréductibles.

  • Facteur linéaire (degré 1) : \((x-a)\). Il produit en éléments simples des termes du type : \[ \frac{A}{x-a},\ \frac{B}{(x-a)^2},\ \dots,\ \frac{K}{(x-a)^n} \] si la racine \(a\) est de multiplicité \(n\).
  • Facteur quadratique irréductible (degré 2) : \(x^2+bx+c\) avec discriminant \(b^2-4c<0\). Pour une puissance \(m\) de ce facteur, les termes en éléments simples sont : \[ \frac{B_0x+C_0}{x^2+bx+c},\ \frac{B_1x+C_1}{(x^2+bx+c)^2},\ \dots,\ \frac{B_{m-1}x+C_{m-1}}{(x^2+bx+c)^m}. \] Les numérateurs sont linéaires (car degré inférieur à 2).

Forme générale de la décomposition

Si

$$Q(x)=\prod_i (x-a_i)^{n_i}\prod_j (x^2+b_jx+c_j)^{m_j},$$

alors

$$ \frac{P(x)}{Q(x)}= \sum_i\sum_{k=1}^{n_i}\frac{A_{i,k}}{(x-a_i)^k} +\sum_j\sum_{k=1}^{m_j}\frac{B_{j,k}x+C_{j,k}}{(x^2+b_jx+c_j)^k}. $$

Cas pratique — comparaison des degrés

  • Si \(\deg P < \deg Q\) : la fraction est propre → on décompose directement.
  • Si \(\deg P \ge \deg Q\) : on commence par une division euclidienne \(P=Q\cdot S + R\) avec \(\deg R < \deg Q\), puis \(\dfrac{P}{Q}=S + \dfrac{R}{Q}\) et on décompose \(\dfrac{R}{Q}\).

Division euclidienne — rappel pratique (méthode verticale)

On écrira à la main la division comme pour des entiers : on aligne les termes, on trouve le premier terme du quotient (terme dominant), on multiplie, on soustrait, etc. Voici des exemples complets avec indication du dividende, du diviseur, du quotient et du reste à chaque étape.

Exemple A (simple, déjà vu) :

Diviser \(x^3 + 2x^2 + 3x + 4\) par \(x^2 + 1\).

Dividende : x³ + 2x² + 3x + 4
Diviseur  : x² + 1

Étape 1 :
  - Terme dominant : x³ ÷ x² = x
  - On calcule x · (x²+1) = x³ + x
  - Soustraction : (x³+2x²+3x+4) - (x³+x) = 2x² + 2x + 4   ← reste_1

Étape 2 :
  - Terme dominant : 2x² ÷ x² = 2
  - On calcule 2 · (x²+1) = 2x² + 2
  - Soustraction : (2x²+2x+4) - (2x²+2) = 3x + 2  ← reste_final

Résultat :
  Quotient Q(x) = x + 2
  Reste R(x) = 3x + 2

Donc :
  $$\frac{x^3 + 2x^2 + 3x + 4}{x^2 + 1} = x + 2 + \frac{3x+2}{x^2+1}.$$

Exemples demandés (cas avec dénominateur complexe)

On considère le dénominateur commun :

$$B(x)=(x^2+1)^2(x+1)^2(x-2).$$

Étudions les deux fractions demandées :

Cas 1 — Numérateur de degré supérieur :

$$\frac{x^8 + 2x^2 + 3x + 4}{(x^2+1)^2(x+1)^2(x-2)}.$$

Étape 1 — division euclidienne :

Dividende (A) : \(A(x)=x^8+2x^2+3x+4\)

Diviseur (B) : \(B(x)=(x^2+1)^2(x+1)^2(x-2)\)

Pour manipuler la division il est pratique d'étendre \(B(x)\) :

$$\displaystyle B_{\text{exp}}(x)=x^7 - x^5 - 2x^4 - 5x^3 - 4x^2 - 3x - 2.$$

On divise maintenant \(A(x)\) par \(B_{\text{exp}}(x)\) :

Dividende A(x)  = x⁸ + 0·x⁷ + 0·x⁶ + 0·x⁵ + 0·x⁴ + 2x² + 3x + 4
Diviseur B(x)   = x⁷ - x⁵ - 2x⁴ - 5x³ - 4x² - 3x - 2

Étape 1 :
  - Terme dominant : x⁸ ÷ x⁷ = x
  - Produit : x · B(x) = x⁸ - x⁶ - 2x⁵ - 5x⁴ - 4x³ - 3x² - 2x
  - Soustraction : A(x) - x·B(x) =
      x⁶ + 2x⁵ + 5x⁴ + 4x³ + 5x² + 5x + 4

Comme le reste a maintenant degré 6 (< 7 = deg B), la division s'arrête.

Donc :
  Quotient Q(x) = x
  Reste R(x) = x⁶ + 2x⁵ + 5x⁴ + 4x³ + 5x² + 5x + 4

On obtient :
  $$\frac{A(x)}{B(x)} = x + \frac{R(x)}{B(x)}.$$

Remarque : ici la division n'a qu'un seul pas parce que \(\deg A - \deg B = 1\).

Étape 2 — Décomposition en éléments simples de la partie propre

On décompose la fraction propre \(\dfrac{R(x)}{B(x)}\). Le schéma de formes à prévoir (vu les facteurs) est :

$$ \frac{R(x)}{(x^2+1)^2(x+1)^2(x-2)} = \frac{A_1x+B_1}{x^2+1} + \frac{A_2x+B_2}{(x^2+1)^2} + \frac{C}{x+1} + \frac{D}{(x+1)^2} + \frac{E}{x-2}. $$

Après multiplication par le dénominateur complet et égalité des polynômes, on résout pour les constantes \(A_i,B_i,C,D,E\).

Résultat (décomposition complète calculée)

Pour ton exemple précis, la décomposition (obtenue en calcul symbolique) s'écrit :

$$ \boxed{ \frac{x^8 + 2x^2 + 3x + 4}{(x^2+1)^2(x+1)^2(x-2)} = x + \frac{3x-9}{10\,(x^2+1)^2} - \frac{19x+13}{100\,(x^2+1)} - \frac{1}{36(x+1)} - \frac{1}{3(x+1)^2} + \frac{274}{225(x-2)} } $$

(Si tu veux, je peux fournir la démonstration du calcul des coefficients terme à terme — elle consiste à multiplier par le dénominateur et égaler coefficients, ou à évaluer en racines simples pour obtenir rapidement certains termes.)

Cas 2 — Numérateur de degré inférieur (donc fraction propre)

$$\frac{x^3 + 2x^2 + 3x + 4}{(x^2+1)^2(x+1)^2(x-2)}$$

Le degré du numérateur (3) est strictement inférieur à celui du dénominateur (7) : pas de division, on décompose directement.

Forme générale prévue :

$$ \frac{x^3 + 2x^2 + 3x + 4}{(x^2+1)^2(x+1)^2(x-2)} = \frac{A x + B}{x^2+1} + \frac{C x + D}{(x^2+1)^2} + \frac{E}{x+1} + \frac{F}{(x+1)^2} + \frac{G}{x-2}. $$

En multipliant par le dénominateur et en identifiant les coefficients, on obtient (résultat calculé) :

$$ \boxed{ \frac{x^3 + 2x^2 + 3x + 4}{(x^2+1)^2(x+1)^2(x-2)} = \frac{x-3}{5\,(x^2+1)^2} + \frac{22x-31}{50\,(x^2+1)} - \frac{5}{9(x+1)} - \frac{1}{6(x+1)^2} + \frac{26}{225(x-2)} } $$

Tu peux vérifier en multipliant les deux membres par le dénominateur et en vérifiant que les coefficients des puissances de \(x\) coïncident.

Méthode pratique pour trouver les constantes

  1. Écrire la forme générale (selon les facteurs irréductibles et leurs puissances).
  2. Multiplier par le dénominateur complet pour obtenir une égalité entre deux polynômes.
  3. Soit :
    • évaluer en \(x=a\) quand \(a\) est une racine simple (pour éliminer beaucoup de termes),
    • ou identifier coefficient par coefficient (décomposer et comparer les coefficients des puissances de \(x\)).
  4. Résoudre le système linéaire obtenu pour les constantes.

Remarque : pour des quadratiques irréductibles, on ne peut pas utiliser la substitution \(x=a\) (racine réelle) ; on utilise l'identification des coefficients ou parfois des méthodes astucieuses (dériver l'identité, etc.).

4. Intégrales trigonométriques

Utiliser les identités trigonométriques pour réduire puissances et produits :

$$\sin^2 x = \frac{1-\cos(2x)}{2},\quad \cos^2 x = \frac{1+\cos(2x)}{2},\quad \sin(2x)=2\sin x\cos x.$$

Exemples (niveau L2)

1) \(\displaystyle \int \sin^2 x\,dx = \frac{x}{2}-\frac{\sin(2x)}{4}+C.\)

2) \(\displaystyle \int \cos^2 x\,dx = \frac{x}{2}+\frac{\sin(2x)}{4}+C.\)

3) \(\displaystyle \int \sin x\cos x\,dx = -\frac{1}{4}\cos(2x)+C.\)

4) \(\displaystyle \int \tan^2 x\,dx = \int (\sec^2 x -1)\,dx = \tan x - x + C.\)

Méthode : si puissances paires → utiliser identités de double angle ; si impaires → isoler un facteur sin ou cos et substituer \(u=\cos x\) ou \(u=\sin x\).
