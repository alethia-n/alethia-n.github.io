Introduction — Intuition géométrique

L'intégrale d'une fonction réelle continue sur un intervalle [a,b] représente l'aire algébrique délimitée par la courbe y = f(x), l'axe des abscisses et les droites x = a et x = b. Cette intuition conduit à la définition rigoureuse par sommes de Riemann et au théorème fondamental du calcul intégral.

1. Définition rigoureuse (somme de Riemann)

Soit f : [a,b] → ℝ bornée. On subdivise l'intervalle en sous-intervalles pour approximer l'aire sous la courbe :

Soit une partition P = { x_0 = a < x_1 < ... < x_n = b }. Chaque sous-intervalle est [x_{i-1}, x_i] et sa longueur Δx_i = x_i - x_{i-1}.

Choisissons un point ξ_i ∈ [x_{i-1}, x_i]. La somme de Riemann est :

S(P,f,ξ) = ∑_{i=1}^{n} f(ξ_i) Δx_i

Si le pas maximal de la partition tend vers 0 et que la limite est indépendante des choix de ξ_i, on définit l'intégrale :

∫_a^b f(x) dx = lim_{||P||→0} S(P,f,ξ)

Remarques

  • f doit être bornée pour que la somme de Riemann soit définie.
  • Si f est continue sur [a,b], l'intégrale existe toujours.
  • Si f(x) ≥ 0, l'intégrale représente une aire réelle, sinon une aire algébrique.

Exemple

Pour f(x) = x² sur [0,1] et une partition uniforme x_i = i/n :

S(P,f,ξ) = ∑_{i=1}^{n} (i/n)² * (1/n) = (1/n³) ∑_{i=1}^{n} i²

Or ∑_{i=1}^{n} i² = n(n+1)(2n+1)/6, donc :

S(P,f,ξ) = (n+1)(2n+1)/(6n²) → 1/3 lorsque n → ∞

Donc : ∫_0^1 x² dx = 1/3

2. Théorème fondamental du calcul (TFI)

Partie 1 : Soit F(x) = ∫_a^x f(t) dt avec f continue. Alors F est dérivable et F'(x) = f(x).

(F(x+h) - F(x))/h = (1/h) ∫_x^{x+h} f(t) dt

Lorsque h→0, le membre de droite tend vers f(x).

Partie 2 : Si F est une primitive de f sur [a,b], alors :

∫_a^b f(x) dx = F(b) - F(a)

3. Intégrales indéfinies — primitives usuelles

Exemple

∫ x³ dx = x⁴/4 + C
  • ∫ x^n dx = x^{n+1}/(n+1) + C (n≠-1)
  • ∫ dx/x = ln|x| + C
  • ∫ sqrt(x) dx = 2/3 x^{3/2} + C
  • ∫ 1/sqrt(x) dx = 2 sqrt(x) + C
  • ∫ 1/x² dx = -1/x + C
  • ∫ e^{ax} dx = (1/a) e^{ax} + C
  • ∫ a^x dx = a^x / ln(a) + C
  • ∫ ln x dx = x ln x - x + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ tan x dx = -ln|cos x| + C
  • ∫ cot x dx = ln|sin x| + C
  • ∫ sec² x dx = tan x + C
  • ∫ csc² x dx = -cot x + C
  • ∫ sec x tan x dx = sec x + C
  • ∫ csc x cot x dx = -csc x + C
  • ∫ sin² x dx = x/2 - sin(2x)/4 + C
  • ∫ cos² x dx = x/2 + sin(2x)/4 + C

4. Techniques principales d’intégration

4.1 Substitution

∫ f(φ(x)) φ'(x) dx = ∫ f(u) du

Exemple

∫ 2x cos(x²) dx

Posons u = x² ⇒ du = 2x dx, donc :
∫ 2x cos(x²) dx = ∫ cos(u) du = sin(u) + C = sin(x²) + C

Exemple

∫ dx / √(1 - x²)

Posons x = sin θ ⇒ dx = cos θ dθ, √(1-x²) = cos θ :
∫ dx / √(1 - x²) = ∫ dθ = θ + C = arcsin(x) + C

Exemple

∫ 2x / (1 + x²) dx

Posons u = 1 + x² ⇒ du = 2x dx :
∫ 2x / (1 + x²) dx = ∫ du / u = ln|u| + C = ln|1 + x²| + C

4.2 Intégration par parties

Formule : ∫ u v' dx = u v - ∫ u' v dx

Exemple

∫ x e^x dx Posons u = x, dv = e^x dx ⇒ u' = 1, v = e^x :
∫ x e^x dx = x e^x - ∫ e^x dx = x e^x - e^x + C

Exemple

∫ ln x dx Posons u = ln x, dv = dx ⇒ u' = 1/x, v = x :
∫ ln x dx = x ln x - ∫ x*(1/x) dx = x ln x - x + C

Exemple

∫ x cos x dx Posons u = x, dv = cos x dx ⇒ u' = 1, v = sin x :
∫ x cos x dx = x sin x - ∫ sin x dx = x sin x + cos x + C

4.3 Décomposition en fractions simples

Pour R(x) = P(x)/Q(x) avec deg P < deg Q, on factorise Q et écrit R comme somme de fractions simples.

Exemple

∫ (2x + 3) / (x(x+1)) dx Décomposition : (2x+3)/(x(x+1)) = A/x + B/(x+1) Identification : A = 3, B = -1
∫ (2x + 3)/(x(x+1)) dx = 3 ln|x| - ln|x+1| + C

Exemple

∫ (5x + 7)/((x+1)(x+2)) dx Décomposition : (5x+7)/((x+1)(x+2)) = A/(x+1) + B/(x+2) Identification : A = 3, B = 2
∫ (5x + 7)/((x+1)(x+2)) dx = 3 ln|x+1| + 2 ln|x+2| + C

5. Intégrales trigonométriques

Intégrales de puissances de sinus et cosinus :

∫ sin^n x dx, ∫ cos^n x dx

Exemple

∫ sin³ x dx = -cos x + cos³ x / 3 + C

Exemple

∫ cos³ x dx = sin x - sin³ x / 3 + C

Intégrales des fonctions trigonométriques usuelles :

  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ tan x dx = -ln|cos x| + C
  • ∫ cot x dx = ln|sin x| + C
  • ∫ sec² x dx = tan x + C
  • ∫ csc² x dx = -cot x + C
  • ∫ sec x tan x dx = sec x + C
  • ∫ csc x cot x dx = -csc x + C
  • ∫ sin² x dx = x/2 - sin 2x /4 + C
  • ∫ cos² x dx = x/2 + sin 2x /4 + C
  • ∫ sin x cos x dx = sin² x / 2 + C

6. Résumé des techniques d’intégration

Voici un rappel des principales méthodes pour calculer des intégrales, avec leurs principes :

  • Intégration par substitution (changement de variable) :

    Si l’intégrande contient une fonction et sa dérivée, on peut poser u = φ(x) et remplacer dx par du pour simplifier l’intégrale :

    ∫ f(φ(x)) φ'(x) dx = ∫ f(u) du
  • Intégration par parties :

    Utilisée lorsque l’intégrande est un produit de fonctions u(x)v'(x). On choisit u à dériver et v' à intégrer :

    ∫ u v' dx = u v - ∫ u' v dx
  • Décomposition en fractions simples :

    Pour une fraction rationnelle P(x)/Q(x) avec deg P < deg Q, on factorise le dénominateur et on écrit la fraction comme somme de fractions simples, plus faciles à intégrer :

    P(x)/Q(x) = Σ A_i / (x - r_i) + …
  • Intégrales trigonométriques :

    Pour des puissances de sinus et cosinus ou des produits, on utilise des identités trigonométriques pour transformer l’intégrale en une forme plus simple :

    ∫ sin^n x dx, ∫ cos^n x dx, ∫ sin^m x cos^n x dx
  • Intégrales de fonctions usuelles :

    Certaines intégrales sont directes, comme :

    • ∫ x^n dx = x^(n+1)/(n+1) + C
    • ∫ e^(ax) dx = e^(ax)/a + C
    • ∫ 1/x dx = ln|x| + C
    • ∫ sin x dx = -cos x + C, ∫ cos x dx = sin x + C

En combinant ces techniques, la majorité des intégrales rencontrées en calcul différentiel peuvent être résolues.
