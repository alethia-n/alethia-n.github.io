Integrals – Exercises (TD 1)

Exercise 1 – Substitution

\[ \int x \cos(x^2) dx \] Solution: Let \(u = x^2\), \(du = 2x dx\) → \[ \int x \cos(x^2) dx = \frac{1}{2} \int \cos(u) du = \frac{1}{2} \sin(u) + C = \frac{1}{2} \sin(x^2) + C \]

Exercise 2 – Integration by Parts

\[ \int x e^{2x} dx \] Solution: \(u=x, dv=e^{2x}dx \Rightarrow du=dx, v=\frac{1}{2}e^{2x}\) \[ \int x e^{2x} dx = x\frac{1}{2} e^{2x} - \int \frac{1}{2} e^{2x} dx = \frac{1}{2} x e^{2x} - \frac{1}{4} e^{2x} + C \]

Exercise 3 – Partial Fractions

\[ \int \frac{3x+5}{x^2+4x+3} dx \] Decompose: \(\frac{3x+5}{x^2+4x+3} = \frac{A}{x+1} + \frac{B}{x+3}\) Solve: \(A=2, B=1\) \[ \int \frac{3x+5}{x^2+4x+3} dx = 2 \ln|x+1| + \ln|x+3| + C \]
